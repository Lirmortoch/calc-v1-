# Simple Calc

A simple calculator for everyday mathematical calculations.

## Description

A Windows Calculator clone built using Vanilla JS (HTML and CSS as well), the mathjs library for parsing expressions, and webpack to bundle JavaScript files.

The calculator has two modes:

* **Regular mode**: simple calculations (multiplication, division, subtraction, addition, etc.)
* **Scientific mode**: additional functions for more complex calculations

## Getting Started

### Dependencies

* [mathjs](https://mathjs.org/);
* [webpack](https://webpack.js.org/);

### Executing program
Clone repository:
```
git clone https://github.com/Lirmortoch/calc-v1-.git
```
Install dependencies:
```
npm install
```
Run the development server:
```
npm run dev
```
